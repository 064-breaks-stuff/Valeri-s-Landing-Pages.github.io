This repository contains 2 landing pages I've built for https://valerifurniture.com/.

Each landing page is optimised for a certain age demographic. Given that the primary audience is a top 40% earning home-owner, between the ages of 25 and 65, I chose to split the overall age demographic into two age groups: 25-45 and 45-65.

This is because, as per market psychology, 25-45 can be classified as early-late millenials, while 45-65 can be classified as early-late millenials. Each age group will benefit from a slightly modified market approach, and hence the two landing pages.

Each landing page is optimised for its particular age group's audience, making full use of the relevant business and audience understanding.
